# Beggining with Angular
Repository with projects for beginners using Angular

### Node

    $ v14.20.0

### Angular 

    $ v15.0.5

### Start

    $ ng serve

### Create component

    $ ng generate component component-name

### Create interface

    $ ng generate interface interface-name
